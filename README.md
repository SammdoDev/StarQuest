# ✨ StarQuest - Ambil Bintang, Hindari Musuh!

**StarQuest** adalah game sederhana buatan **Samdo Dev** menggunakan **Phaser 3**.  
Pemain mengontrol **karakter manusia** untuk **mengambil bintang** sebanyak mungkin dan **menghindari musuh** yang datang dari mana saja.

## 🎮 Cara Main

- Tekan tombol panah **Kanan (⭢)** untuk bergerak ke kanan  
- Tekan tombol panah **Kiri (⭠)** untuk bergerak ke kiri  
- Tekan tombol **Atas (⭡)** untuk melompat  
- Ambil sebanyak mungkin **bintang** ✨  
- Hindari **musuh** 👾 yang bergerak dari mana saja  
- Jika tertabrak musuh → **Game Over**
